## Getting Started

### Prerequisites

Ensure you have Node.js installed on your system. You can download it from [Node.js website](https://nodejs.org/).

Also, you will need to install Expo CLI:

```bash
npm install -g expo-cli
```
1: **Start project**
```bash
expo start
```
